bool xl7::graphics::meshes::VertexBufferUpdater::write ( const VertexBufferWrite &  write)

Updates a contiguous range of vertices in the vertex buffer, and returns true if the update succeeded, false otherwise.

The input data must match the size of the specified range, taking into account the vertex stride, ensuring that only whole vertices are written.

On success, the corresponding vertex range is marked as "dirty".
