◆ hide_cursor()

int cl7::platform::user::hide_cursor ( )

Decrements the internal display counter that determines whether the mouse cursor should be displayed by one. The cursor is displayed if the display counter is greater than or equal to 0. If a mouse is installed, the initial display count is 0. If no mouse is installed, the display count is -1. Returns the new display counter.
