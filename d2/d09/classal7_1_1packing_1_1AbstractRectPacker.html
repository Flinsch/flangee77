◆ insert() [1/2]

std::vector< std::optional< Rect > > al7::packing::AbstractRectPacker::insert ( const std::vector< Size > &  item_sizes,
RectOptions  options = {} 
)

Tries to insert items of the specified sizes in an offline batch mode. For each requested item size, an optional rectangle is returned, in the corresponding order. However, it is (deliberately) not guaranteed that "earlier" (or even all) rectangles will be considered before later rectangles. In case of doubt, the decision rests with the specific algorithm used.
