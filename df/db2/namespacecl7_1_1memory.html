Calculates the largest reasonable unit for a byte count and scales the value accordingly. You can specify the base factor for scaling (e.g., 1000 for decimal SI, 1024 for binary IEC) and the maximum number of orders of magnitude to consider. The actual number of orders of magnitude is optionally stored via the corresponding output parameter.
