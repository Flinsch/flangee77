Detailed Description

template<class TAstNode = AstNode>
class dl7::syntax::DirectAstBuilder< TAstNode >

An abstract AST builder responsible for constructing an abstract syntax tree (AST) directly from a sequence of tokens, bypassing the construction of a concrete syntax tree (CST). An AST builder is an intermediate stage after tokenization and before AST processing. It analyzes a sequence of tokens and builds a compact and semantically meaningful AST representation.

This serves as an alternative to the traditional two-step approach, where a CST is first built and then transformed into an AST. Direct AST construction can be beneficial for performance or when a CST is not necessary.
