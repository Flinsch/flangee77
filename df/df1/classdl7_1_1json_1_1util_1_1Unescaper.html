Detailed Description

Unescapes a JSON-escaped string literal (excluding surrounding quotes). Processes escape sequences such as \n, \t, \uXXXX, etc., converting them to their literal character equivalents. Even if it's not valid JSON, things like \xXX or \v are still taken into account. Otherwise assumes valid UTF-8 encoding and JSON string syntax.
