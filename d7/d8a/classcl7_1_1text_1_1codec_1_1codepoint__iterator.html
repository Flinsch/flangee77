Detailed Description

template<cl7::any_char Tchar, class TDefaultErrorHandler = DefaultErrorHandler>
requires (std::is_default_constructible_v<TDefaultErrorHandler>)
class cl7::text::codec::codepoint_iterator< Tchar, TDefaultErrorHandler >

Provides random access to the sequence of (Unicode) code points within a string. It abstracts away the underlying encoding (e.g., ASCII, UTF-8, UTF-16, etc.) and allows iteration, indexing, and arithmetic operations directly on code points, not code units.
