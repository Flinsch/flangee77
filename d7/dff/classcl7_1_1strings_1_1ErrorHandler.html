CodepointResult cl7::strings::ErrorHandler::check_adjust_ascii ( codepoint  codepoint) const
inline

Checks whether the given code point represents a valid ASCII character with a value of up to 0x7F and, if not, replaces it with the ASCII replacement symbol. Returns an error code if applicable or none/zero if valid.

◆ check_adjust_unicode()

CodepointResult cl7::strings::ErrorHandler::check_adjust_unicode ( codepoint  codepoint) const
inline

Checks whether the given code point is valid Unicode with a maximum value of U+10FFFF and not within the surrogate range U+D800 through U+DFFF, and if not, replaces it with the Unicode replacement symbol. Returns an error code if applicable or none/zero if valid.
