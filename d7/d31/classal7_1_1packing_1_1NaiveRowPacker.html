Detailed Description

A simple 2D bin packer that places rectangular items sequentially in horizontal rows. Each new item is placed at the current row's cursor position (x, y). The cursor advances to the right by the item's width. If the item does not fit within the current row, a new row is started at x = 0 and y is advanced by the maximum item height of the previous row.

This naive approach is essentially the same as the Next-Fit (NF) heuristic of a Shelf packer.

Pretty fast, as it doesn't perform any searching or backtracking, but may produce significant unused space with items of varying heights and/or with wide items that start a new row.
