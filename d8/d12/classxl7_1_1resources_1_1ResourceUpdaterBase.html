template<class TResourceDesc , class TDirtyState >
bool xl7::resources::ResourceUpdaterBase< TResourceDesc, TDirtyState >::write_full ( cl7::byte_view  data)
inline

Replaces the entire contents of the internal buffer with the provided data. Returns true if the update succeeded, false otherwise.

This is a convenience function for full-buffer updates. The size of the provided data must exactly match the size of the underlying buffer.

On success, the internal buffer is marked as "fully dirty".
