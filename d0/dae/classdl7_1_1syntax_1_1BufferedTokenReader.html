Detailed Description

template<class Tcontainer = std::vector<Token>>
class dl7::syntax::BufferedTokenReader< Tcontainer >

A token reader that retrieves tokens from a pre-generated sequence. This implementation stores tokens in an internal buffer (such as an std::vector) and allows sequential reading with lookahead support.
